# 简易聊天室网站

该项目提供一个基于 `Express` 与 `Socket.IO` 的实时聊天室，支持多人在线发送消息，前端界面采用原生 HTML/CSS/JavaScript 实现。

## 功能特性

- 实时双向通信：用户消息即时广播给所有在线用户。
- 内置机器人：自动回复常见问候、天气、时间等话题，可模拟与机器人聊天。
- 系统提示：用户加入或离开时显示系统消息。
- 输入校验与限制：昵称与消息长度限制、防止发送空消息。
- 响应式界面：在桌面端与移动端均具备良好体验。


## 环境准备

1. 安装 [Node.js](https://nodejs.org/)（建议 18+ 版本）。
2. 在项目根目录执行依赖安装：

```
npm install
```

## 本地运行

开发模式（自动重启）：

```
npm run dev
```

生产模式：

```
npm start
```

启动后访问浏览器中的 `http://localhost:3000`。

## 机器人互动提示

- 初次进入页面时，机器人会主动打招呼，后续会针对你发送的消息生成回复。
- 支持识别问候、感谢、天气、时间等关键词，也会对包含问号的问题作出反馈。
- 若想停止对话，可发送 “再见”“拜拜” 等词语，机器人会礼貌结束。

## 部署建议

- 可将项目部署至支持 Node.js 的平台（如 Vercel、Render、Railway、阿里云函数计算等）。
- 建议通过反向代理（Nginx/Apache）或平台提供的 HTTPS 支持启用 TLS。
- 如需持久化聊天记录或用户管理，可接入数据库（例如 MongoDB、PostgreSQL）并扩展服务端逻辑。
